# Student Performance Dataset

## Dataset
Student Performance dataset is a tabular dataset stored in CSV format.

## Description
The dataset contains information about students and their academic performance.
It includes personal and academic features such as age, gender, study time, absences, and exam scores.

## Machine Learning Problem Type
Classification.

## Target Variable
G3 (Final grade of the student).

## Goal
The goal of this dataset is to analyze and classify students based on their final academic performance.

## Tools Used
Python, pandas, Jupyter Notebook.
